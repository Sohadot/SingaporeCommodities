# Privacy Policy

SingaporeCommodities.com operates under a principle of **minimal data surface**.

We do not build value through user surveillance, profiling, or data extraction.  
The platform is designed as a **knowledge infrastructure**, not a data-harvesting system.

## Data Collection

We collect only what is strictly necessary for:
- Platform stability and security
- Aggregate traffic understanding
- Performance monitoring

This may include:
- Anonymous usage data
- Basic request logs
- Non-identifiable behavioral signals

No personal profiling is conducted.

## Cookies

If used, cookies are strictly:
- Functional (site performance)
- Analytical (aggregate, non-identifiable)

No third-party advertising trackers are embedded.

## Data Usage

Collected data is used exclusively to:
- Improve site performance
- Maintain system reliability
- Understand structural usage patterns

It is not sold, traded, or shared for commercial targeting.

## External Services

Where analytics tools are used, they are configured to:
- Minimize tracking
- Avoid personal identification
- Respect jurisdictional privacy standards

## Philosophy

This platform is built on a simple premise:

> Trust is not declared. It is engineered.

SingaporeCommodities.com prioritizes **structural trust over extractive growth models**.
